Lisp Documentation
Code by Noah Cassidy, unless otherwise indicated.
This page includes: A complete list of commands, documentation of all functions, and an index. Click the command name to jump to the documentation.
Big Process
These commands are used during the labelling and aligning processes. Use of these commands outside these specific cases is not recommended.
AEXR
Auto label rooms and export their labels
NEXR
Auto export room labels (like AEXR), but don't overwrite monument numbers
BEGIN
Check drawing alignment
COPYACCESS
Copy the contents of the "Access" tab
COPYADDR
Copy the attributes of the title block between 8.5x11 and 11x17 layouts.
ENTRANCE_IDS
Move all entrance labels to model space
EXPORTNEWIDS
Export and find room ids and labels, assign new monument ids to the rooms
EXR
Shortcut for EXPORTNEWIDS
FAXESS
Fix the accessibility drawing
FAX
Shortcut for FAXESS
FIXERR
Detect errors for SMN
FIXROOM
Find broken net area polygons
LABELLAYER
Add monument ids to a specific layer
LL
Shortcut for LABELLAYER
PASTEACCESS
Align the access drawing to an sdf drawing
PASTEASPECTACCESS
Earlier iteration of PASTEACCESS
PASTENOROTATE
Insert the access drawing, but will not auto align
PM
Shortcut for PREPMERGE
PREPMERGE
Clean up a drawing before merging it.
PREPALIGN
Clean up an sdf drawing before aligning it
ROOF
Run the gui to label room sections.
SCNMOVE
Scale objects from 8.5x11 to 11x17 drawing
SMN
Label all monument numbers
STOP
Detaches "All Site Plans" xref
Utility
These Commands accomplish smaller, more general tasks than the "Big Process" commands. They do everything from changing units to checking for certain errors. However they are likely not useful unless developing lisp, or doing similar tasks to the "Big Processes".
ATTDEF2TEXT
Turn attdef entities into text objects
BATCHLISP
Run a lisp script on several files
CHGUNITS
Change units to inches
E2A
Convert ellipses to arcs
ENAME
Return the entity name of a selected object
EPROP
Ouput the entity properties to the console
EXPORTROOMIDS
Export all rooom ids to export3.csv
ISOM
Isolate the misc layer
PORTPROP
Add property portable label text to the layout
STRACE
Add "Site Plan Trace" xref
TYPES
Ouput the type of the object to the console
MER
Outline a selection of objects with a rectangle
Useful
These commands are utilities that are useful more generally, no matter what the current task is, although some are specific to monument ids.
CS
Close and save the drawing
FAC_ID
Add facility id to all objects
IDS
"short"cut for FAC_ID
FC
Shortcut for FIXCIRCLES
FIXCIRCLES
Turn circles into polylines
GETCONFIG
Get the value stored in "H:\\Map 3d\\config.txt" (the preferred OSNAP setting)
MERGECAD
Import selected drawings as blocks on the current layer
MERGEFLOORS
Import selected drawings as blocks, seperating each onto the appropriate layer
MERGEXCAD
Import selected drawings exploded onto the current layer
MPO
Shorcut for MPOINT
MPOINT
Place a point objects between two points
NEWFIELDS
Adds the facility number in site_id object data
REPLACECAD
Replace each same-named drawing with the newly selected file
RFC
Remove polyface meshes
RSNAP
Reset OSNAP setting to whatever is in "H:\\Map 3d\\config.txt"
SETCONFIG
Set "H:\\Map 3d\\config.txt" to the preferred OSNAP setting
UNDOALL
Reset a file back to the state when it was first opened
UPDATELAYERS
Add layers to update to new standard
UPLAY
Shortcut for UPDATELAYERS
XREFFLOORS
Add all lower floors as an xref to the current drawing
Not Useful
These are very specific commands, likely not useful outside of their specific script implementations, and only listed here to provide a complete list of commands included in the library.
BOUNDFIXROOM
find errors in a merged file with findROOMerr
CATCH22
Find all files that still contain the "22" layer
COLORLINES
Old debug/testing command
FIXALL
Fix all ids in a selection set after they were all set to use facility number 0388 rather than the correct one
